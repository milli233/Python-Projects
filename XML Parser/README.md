# XML Parser Project

## Description
This project parses an XML file containing book information and extracts details such as **Book Title, Author, and Issued Date**. It demonstrates XML parsing techniques using Python and helps in handling structured data efficiently.

## Features
- Parse XML files to extract book details
- Handle missing tags gracefully
- Count unique books and authors
- Display issued dates
